Implemented functionality overview

Google Sign in

Use your Google account to sign in.

Pen size

Pen size varies from 1 to 4.

Tools

There is a pen, eraser, stroke line, pipette, bucket, magic bucket.

Palette

Palette with primary and secondary colors and the ability to swap them.

Frame section

Frame section with ability to add/delete a frame, reorder a frame via Drag and Drop, duplicate a frame.

Animation preview

Preview your animation in real time, full screen mode and the ability to change the speed of the animation.

Resize canvas

Canvas size can be changed. Available sizes 32x32, 64x64, 128x128.

Export

You can export created animations as gif files, as well as scale before export.

Keyboard shortcuts

Most actions are accessible from the keyboard (tools, swap colors, add a frame, full-screen animation). There is a modal window for changing keyboard shortcuts

Hover tips

Tool information and keyboard shortcuts are available on hover
